# Music-Player Music升级版

### 项目安装
```
npm install
```

### 项目部署
```
npm run serve
```

### 项目打包
```
npm run build
```

与前版相比页面构造更加美观、更多评论获取、播放器升级、音乐分页搜索、新增播放列表
